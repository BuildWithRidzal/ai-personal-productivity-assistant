# ChatinMe - AI Personal Productivity Assistant

ChatinMe adalah chatbot berbasis AI menggunakan Streamlit dan Gemini API untuk membantu pengguna membuat jadwal, memecah tugas besar, menentukan prioritas, membuat catatan, dan mengelola to-do list.

## Features

- AI chatbot dengan Gemini API
- Daily planner assistant
- Task breakdown assistant
- Priority coach
- Notes workspace
- To-do list manager
- File upload support
- Voice input support
- Light/Dark mode

## Tech Stack

- Python
- Streamlit
- Google Gemini API
- google-genai
- python-dotenv
